# 1、剑指 Offer 09. 用两个栈实现队列 (opens new window)

# 题目

用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 )

示例 1：

输入： ["CQueue","appendTail","deleteHead","deleteHead"] [[],[3],[],[]] 输出：[null,null,3,-1] 示例 2：

输入： ["CQueue","deleteHead","appendTail","appendTail","deleteHead","deleteHead"] [[],[],[5],[2],[],[]] 输出：[null,-1,null,null,5,2] 提示：

1 <= values <= 10000 最多会对 appendTail、deleteHead 进行 10000 次调用

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/yong-liang-ge-zhan-shi-xian-dui-lie-lcof 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

# 分析

栈的主要特点是：先进后出

队列的主要特点是：先进先出

两个栈，一个栈实现插入操作，一个栈实现删除操作

先往第一个栈里面进行插入元素，第一个插入的元素在栈的底部，最后插入的在栈的顶部，当第一个栈要删除的时候，把第一个栈要删除的元素依次插入第二个栈中（第二个栈为空），第二个栈中的元素就是待删除的元素顺序，依次弹出第二个栈中的元素即可！

# 实现

class CQueue {
    //定义两个栈
    Deque<Integer> stack1;
    Deque<Integer> stack2;

    public CQueue() {
        stack1 = new LinkedList<Integer>();
        stack2 = new LinkedList<Integer>();
    }
    //第一个栈插入元素
    public void appendTail(int value) {
        stack1.push(value);
    }
    
    //第二个栈删除元素
    public int deleteHead() {
        if (stack2.isEmpty()){
            while(!stack1.isEmpty()){
                stack2.push(stack1.pop());
            }
        }
        //当第二个栈不为空时，依次弹出第二个栈中元素
        if(stack2.isEmpty()){
            return -1;
        }else{
            int item = stack2.pop();
            return item;
        }
    }
}

# 2、包含min函数的栈 (opens new window)

# 题目

定义栈的数据结构，请在该类型中实现一个能够得到栈的最小元素的 min 函数在该栈中，调用 min、push 及 pop 的时间复杂度都是 O(1)。

示例:

MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.min();   --> 返回 -3.
minStack.pop();
minStack.top();      --> 返回 0.
minStack.min();   --> 返回 -2.

# 分析

  • **思路一：**新建一个栈B来存储栈中的最小元素
    • 栈A每次push元素后，判断栈B中元素是否为空或者B栈顶元素是否大于新元素，如是，则Bpush该元素
    • pop()出栈，则需要保持A、B的一致性，A出栈则是A.pop()，若是A出栈的元素.equals(B出栈的元素)，则B也需要出栈
      • 这样做的原因：若是只有A出栈，B不出栈的话，则无法保证A与B元素的同步
    • A栈顶的元素则是A.peek()
    • 最小的元素则是B.peek()

# 实现

//思路一：
class MinStack {
    
    // stack1存储元素，stack2存储stack1中最小的元素
    Stack<Integer> stack1,stack2;

    /** initialize your data structure here. */
    public MinStack() {
        stack1 = new Stack<>();
        stack2 = new Stack<>();
    }
    
    public void push(int x) {
        stack1.add(x);
        if(stack2.isEmpty() || stack2.peek() >= x){
            stack2.push(x);
        }
    }
    
    public void pop() {
        if (stack1.peek().equals(stack2.peek())){
            stack2.pop();
        }
        stack1.pop();
    }
    
    public int top() {
        return stack1.peek();
    }
    
    public int min() {
        return stack2.peek();
    }
}

# 题目

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

示例 1：

输入：head = [1,3,2]
输出：[2,3,1]

# 方法一：递归

将链表的值递归的增加到list中

class Solution {
    //list用来接收链表的值
    List<Integer> list = new ArrayList<>();
    public int[] reversePrint(ListNode head) {
        resverListNode(head);

        return result;

    public void resverListNode(ListNode head){
        //为空 则返回
        if (null == head)
        return;
        resverListNode(head.next);
        list.add(head.val);
    }
}

# 方法二：栈

栈的原理：先进后出

实现原理就是把链表节点一个个入栈， 当全部入栈完之后再一个个出栈， 出栈的时候在把出栈的结点串成一个新的链表。

class Solution {

    public int[] reversePrint(ListNode head) {
        //利用栈的先进后出
    LinkedList<Integer> stack = new LinkedList<Integer>();
        while (null != head){
            stack.addLast(head.val);
            head = head.next;
        }
        int[] result = new int[stack.size()];
        for(int i = 0; i < result.length; i ++){
            result[i] = stack.removeLast();
        }

        return result;
    }
}

# 2、剑指 Offer 24. 反转链表 (opens new window)

# 题目

定义一个函数，输入一个链表的头节点，反转该链表并输出反转后链表的头节点。

示例:

输入: 1->2->3->4->5->NULL 输出: 5->4->3->2->1->NULL

限制：

0 <= 节点个数 <= 5000

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/fan-zhuan-lian-biao-lcof 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

# 分析与实现

# 方法一：双链表

把原来的节点一个一个摘掉，每次摘掉的链表都让它成为新的链表的头结点， 然后更新新链表。

class Solution {
    ListNode newListNode = null;
    public ListNode reverseList(ListNode head) {
        while(null != head){
            //获取当前节点的下一节点
            ListNode nextNode = head.next;
             //每次访问的原链表节点都会成为新链表的头结点
            head.next = newListNode;
            //更新新链表
            newListNode = head;
            //重新赋值
            head = nextNode;
        }
        return newListNode;
    }
}

# 方法二：栈

链表的每一个节点push进栈中，然后依次弹出pop()组成新的链表

class Solution {
    
    public ListNode reverseList(ListNode head) {
        Stack<ListNode> stack = new Stack<>();
        while(null != head){
            //链表入栈
            stack.push(head);
            head = head.next;
        }
        if (stack.isEmpty()){
            return null;
        }
        //弹出头节点
        ListNode rehead = stack.pop();
        //初始化结果
        ListNode result = rehead;
        while(!stack.isEmpty()){
            ListNode temp = stack.pop();
            //头节点的下一节点
            rehead.next = temp;
            //重新复制
            rehead = rehead.next;
        }
        rehead.next =null;
        return result;
            
    }
}

# 方法三：递归

递归遍历当前节点的下一节点，然后把下一节点的下一节点复制给当前节点，当前节点的下一节点指向null

class Solution {
    
    public ListNode reverseList(ListNode head) {
        //为空直接返回head
        if (null == head || null == head.next){
            return head;
        }
        //递归遍历
        ListNode result = reverseList(head.next);
        //重新赋值给head
        head.next.next = head;
        //head 的下一个节点为空
        head.next = null;
        return result;  
    }
}

# 1、剑指 Offer 05. 替换空格 (opens new window)

# 题目

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

示例 1：

输入：s = "We are happy." 输出："We%20are%20happy."

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/ti-huan-kong-ge-lcof 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

# 分析与实现

  • 将字符串转换为字符
  • 遍历每一个字符
  • 若是空格，则替换为“%20”不是空格，则增加原字符
    • 实现一：用StringBuilder
    • 实现二：用char字符增加，注意：替换的时候，一个空格对应三个字符，初始化大小应注意
//用StringBuilder
class Solution {
    public String replaceSpace(String s) {
        StringBuilder sb = new StringBuilder();
        for(Character c : s.toCharArray()){
            if (' ' == c){
                sb.append("%20");
            }else{
                sb.append(c);
            }
        }
        return sb.toString();
    }
}

//采用char字符处理
class Solution {
    public String replaceSpace(String s) {
        //获取字符串的长度
        int length = s.length();
        char[] arr = new char[3 * length];
        int size = 0;
        for(int i = 0; i < length; i++){
            char c = s.charAt(i);
            if (' ' == c){
                arr[size++] = '%';
                arr[size++] = '2';
                arr[size++] = '0';
            }else{
                arr[size++] = c;
            }
        }
        String result = new String(arr, 0, size);
        return result;
    }
}

# 2、剑指 Offer 58 - II. 左旋转字符串 (opens new window)

# 题目

字符串的左旋转操作是把字符串前面的若干个字符转移到字符串的尾部。请定义一个函数实现字符串左旋转操作的功能。比如，输入字符串"abcdefg"和数字2，该函数将返回左旋转两位得到的结果"cdefgab"。

示例 1：

输入: s = "abcdefg", k = 2 输出: "cdefgab" 示例 2：

输入: s = "lrloseumgh", k = 6 输出: "umghlrlose"

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/zuo-xuan-zhuan-zi-fu-chuan-lcof 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

# 分析与实现

方法一：最蠢的办法，通过遍历字符串，获取小于k的字符串以及大于等于的k 字符串，然后两者拼接即可

方法二：采用字符串分割的函数，substring()

方法三：方法一的简化版，原理是一样的，采用StringBuilder

方法四：简化版字符串拼接

方法五：我个人觉得有点秀，反正我是想不到，采用取余的操作，没错取余%

//1.最蠢的办法，遍历字符串
class Solution {
    public String reverseLeftWords(String s, int n) {
        int length = s.length();
        char[] ar = new char[n];
        char[] ar1 = new char[length - n];
        int size = 0;
        for(int i = 0; i < length; i++){
            if (i < n){
                ar[size++] = s.charAt(i);
            }else{
                ar1[i-size] = s.charAt(i);
            }
          
        }
        String result = new String(ar, 0, size);
        result = new String(ar1, 0, length-size).concat(result);
        return result;
    }
}

//2.采用substring
class Solution {
    public String reverseLeftWords(String s, int n) {
        return s.substring(n, s.length()) + s.substring(0, n);
    }
}

class Solution {
    public String reverseLeftWords(String s, int n) {
        String res = "";
        for(int i = n; i < s.length(); i++)
            res += s.charAt(i);
        for(int i = 0; i < n; i++)
            res += s.charAt(i);
        return res;
    }
}
//方法五：取余操作，比较秀 方法4的简化版
class Solution {
    public String reverseLeftWords(String s, int n) {
        String res = "";
        for(int i = n; i < n + s.length(); i++)
            res += s.charAt(i % s.length());
        return res;
    }
}

# 4、查找算法

# 1、剑指 Offer 03. 数组中重复的数字 (opens new window)

# 题目

找出数组中重复的数字。

在一个长度为 n 的数组 nums 里的所有数字都在 0～n-1 的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。

示例 1：

输入： [2, 3, 1, 0, 2, 5, 3] 输出：2 或 3

来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/shu-zu-zhong-zhong-fu-de-shu-zi-lcof 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

# 分析与实现

  • 方法一：双层for循环遍历

  • 方法二：先排序，再判断

  • 方法三：用set存储元素

  • 方法四：原地交换

    • 在一个长度为 n 的数组 nums 里的所有数字都在 0 ~ n-1 的范围内，说明数组的下标索引和元素是一对多的关系

    • 然后我们通过交换，变成一对一的关系

    • 遍历中，第一次遇到数字 x 时，将其交换至索引 x 处；而当第二次遇到数字 xx时，一定有 nums[x] = x ，此时即可得到一组重复数字

    • （图片来自） (opens new window)

//双层for循环遍历 
// 时间复杂度 O(n^2)
class Solution {
    public int findRepeatNumber(int[] nums) {
        int result = 0;
        loop:for(int i = 0; i < nums.length; i++){
            for(int j = i + 1; j < nums.length; j++){
                if(nums[i] == nums[j]){
                    result = nums[i];
                    break loop; 
                }
            }
        }
        return result;
    }
}
//先排序，再判断
class Solution {
    public int findRepeatNumber(int[] nums) {
        Arrays.sort(nums);
        int result = 0;
        for(int i = 0; i < nums.length - 1; i++){
            if (nums[i] == nums[i + 1]){
                result = nums[i];
            }
        }
        return result;
    }
}
//用set存储元素
class Solution {
    public int findRepeatNumber(int[] nums) {
        Set<Integer> set = new HashSet<Integer>();
        for(int i = 0; i < nums.length; i++){
            if(set.contains(nums[i])){
                return nums[i];
            }else{
                set.add(nums[i]);
            }
        }
        return -1;
    }
}
//原地交换
class Solution {
    public int findRepeatNumber(int[] nums) {
        int i = 0;
        while (i < nums.length){
            //当前值与下标相等，则下标+1
            if (nums[i] == i){
                i++;
                continue;
            }
            //如果存储下标的值与值相等 ，则返回当前值,否则交换元素
            if(nums[nums[i]] == nums[i]){
                return nums[i];
            }else{
                int temp = nums[i];
                nums[i] = nums[temp];
                nums[temp] = temp;
            }
        }
        return -1;
    }
}
