# 1、二叉树的前序遍历

# ❓题目描述

给你二叉树的根节点 root ，返回它节点值的 前序 遍历。 数据范围：二叉树的节点数量满足 1≤n≤100 ，二叉树节点的值满足1≤val≤100 ，树的各节点的值各不相同

# 👉题目示例

示例1 

输入：{1,#,2,3}
返回值：[1,2,3]
1
2

# 💡题目解析

先访问根结点，然后再访问左子树，最后访问右子树

# 📝代码实现

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 *   public TreeNode(int val) {
 *     this.val = val;
 *   }
 * }
 */

public class Solution {
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     *
     * @param root TreeNode类
     * @return int整型一维数组
     */
    List<Integer> list = new ArrayList<>();
    public int[] preorderTraversal (TreeNode root) {
        // 为空，返回空
        if (null == root) {
            return new int[] {};
        }
        //增加根节点
        list.add(root.val);
        //遍历左子树节点
        if (null != root.left) {
            preorderTraversal(root.left);
        }
        //遍历右子树节点
        if (null != root.right) {
            preorderTraversal(root.right);
        }
        //返回的结果
        int[] res = new int[list.size()];
        for (int i = 0; i < list.size(); i++)
            res[i] = list.get(i);
        return res;
    }
}

# 2、二叉树的中序遍历

# ❓题目描述

给定一个二叉树的根节点root，返回它的中序遍历结果。 数据范围：树上节点数满足 0≤n≤1000，树上每个节点的值满足 −1000≤val≤1000

# 👉题目示例

示例1 

输入：{1,2,#,#,3}
返回值：[2,3,1]
1
2

示例2

输入：{}
返回值：[]
1
2

示例3 

输入：{1,2}
返回值：[2,1]
1
2

示例4 

输入：{1,#,2}
返回值：[1,2]
1
2

# 💡题目解析

照样可以按照递归去解决，这次换一种思路，栈，栈的原理是先进后出 中序遍历：先访问根结点，然后再访问左子树，最后访问右子树 用栈解决的思路：先把左子树节点全部入栈，然后弹出栈顶节点，并把值加入到list中，最后把弹出节点的右子树节点赋值给根节点

# 📝代码实现

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 *   public TreeNode(int val) {
 *     this.val = val;
 *   }
 * }
 */

public class Solution {
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param root TreeNode类 
     * @return int整型一维数组
     */
     
    public int[] inorderTraversal (TreeNode root) {
        // 初始化
        List<Integer> list = new ArrayList<>();
        Stack<TreeNode> s = new Stack<TreeNode>();
        //节点为空返回
        if(null == root){
            return new int[0];
        }
        //当根节点不为空，且栈不为空
        while(null != root || !s.isEmpty()){
            //访问左子树节点
            while(null != root){
                s.push(root);
                root = root.left;
            }
            //弹出当前节点的值
            TreeNode temp = s.pop();
            list.add(temp.val);
            //访问右子树节点
            root = temp.right;
        }
        //初始化返回结果
        int[] res = new int[list.size()];
        for(int i = 0; i < list.size(); i++){
            res[i] = list.get(i);
        }
        return res;
    }
}

# 3、二叉树的后序遍历

# ❓题目描述

给定一个二叉树，返回他的后序遍历的序列。 后序遍历是值按照 左节点->右节点->根节点 的顺序的遍历。

# 👉题目示例

示例1 

输入：{1,#,2,3}
返回值：[3,2,1]
1
2

示例2

输入：{1}
返回值：[1]
1
2

# 💡题目解析

同样，我们采用栈的思路来解，前面已经用过中序遍历来解这个题了， 后续遍历：先访问左子树节点，在访问右子树节点，最后访问根节点 前面跟中序遍历还是很类型的，先访问左子树节点，全部入栈之后；弹出栈顶元素，判断这个根的右边有没有节点或是有没有被访问过，如果没有右节点或是被访问过了，可以访问这个根，并将前序节点标记为这个根；如果没有被访问，那这个根必须入栈，进入右子树继续访问，只有右子树结束了回到这里才能继续访问根。

# 📝代码实现

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 *   public TreeNode(int val) {
 *     this.val = val;
 *   }
 * }
 */

public class Solution {
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param root TreeNode类 
     * @return int整型一维数组
     */
    public int[] postorderTraversal (TreeNode root) {
        //定义list和栈
        List<Integer> list = new ArrayList<Integer>();
        Stack<TreeNode> stack = new Stack<>();
        //根节点为空直接返回
        if(null == root){
            return new int[0];
        }
        //用于记录是否已经遍历访问
        TreeNode pre = null;
        while(null != root || !stack.isEmpty()){
            //访问左子树节点
            while(null != root){
                stack.push(root);
                root = root.left;
            }
            //弹出栈顶元素
            TreeNode temp = stack.pop();
            //判断是否存在右子树节点
            if(null == temp.right || temp.right == pre){
                list.add(temp.val);
                pre = temp;
            }else{
                //不为空，访问右子树节点
                stack.push(temp);
                root = temp.right;
            }
        }
        int[] res = new int[list.size()];
        for(int i = 0; i < list.size(); i++){
            res[i] = list.get(i);
        }
        return res;
    }
}

# 4、 求二叉树的层序遍历

# ❓题目描述

给定一个二叉树，返回该二叉树层序遍历的结果，（从左到右，一层一层地遍历）

# 👉题目示例

示例1

输入：{1,2}
返回值：[[1],[2]]
1
2

示例2

输入：{1,2,3,4,#,#,5}
返回值：[[1],[2,3],[4,5]]
1
2

# 💡题目解析

层序遍历，就是从根节点（第一层）开始，依次向下，获取每一层所有结点的值

  • 创建队列，存储每一层的结点； 使用循环从队列中弹出一个结点：
  • 获取当前结点的key
  • 如果当前结点的左子结点不为空，则把左子结点放入到队列中
  • 如果当前结点的右子结点不为空，则把右子结点放入到队列中

# 📝代码实现

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 * }
 */

public class Solution {
    /**
     * 
     * @param root TreeNode类 
     * @return int整型ArrayList<ArrayList<>>
     */
    public ArrayList<ArrayList<Integer>> levelOrder (TreeNode root) {
        //如果为空，返回空数组
        ArrayList<ArrayList<Integer>> res = new ArrayList<>();
        if(null == root){
            return res;
        }
        //队列存储，进行层次遍历
        Queue<TreeNode> node = new ArrayDeque<TreeNode>();
        node.add(root);
        while(!node.isEmpty()){
		    //存储每一层节点
             ArrayList<Integer> row = new ArrayList<>();
             int n = node.size();
             for(int i = 0; i < n; i++){
                 TreeNode temp = node.poll();
                 row.add(temp.val);
                 if(null != temp.left){
                     node.add(temp.left);
                 }
                 if(null != temp.right){
                     node.add(temp.right);
                 }
            }
            res.add(row);
        }
        return res;
    }
}

# 5、按之字形顺序打印二叉树

# ❓题目描述

给定一个二叉树，返回该二叉树的之字形层序遍历，（第一层从左向右，下一层从右向左，一直这样交替） 数据范围：0≤n≤1500,树上每个节点的val满足 ∣val∣<=1500

# 👉题目示例

示例1 

输入：{1,2,3,#,#,4,5}
返回值：[[1],[3,2],[4,5]]
说明：如题面解释，第一层是根节点，从左到右打印结果，第二层从右到左，第三层从左到右。
1
2
3

示例2

输入：{8,6,10,5,7,9,11}
返回值：[[8],[10,6],[5,7,9,11]]
1
2

示例3

输入：{1,2,3,4,5}
返回值：[[1],[3,2],[4,5]]
1
2

# 💡题目解析

思路1：队列 很明显，就上一题的层次遍历类似，只不过打印的顺序发生了变化，第一层是根节点，从左到右打印结果，第二层从右到左，第三层从左到右。 那么可以定义一个boolean类型的变量，用来判断是奇数行还是偶数行，奇数行进行反转

思路2：栈 思路1中用到了反转，反转可以想到什么，不就是先进后出吗，那么可以使用栈进行处理

  • 首先判断二叉树是否为空，空树没有打印结果
  • 建立两个辅助栈，每次依次访问第一个栈s1与第二个栈s2，根节点先进入s1.
  • 依据依次访问的次序，s1必定记录的是奇数层，访问节点后，将它的子节点（如果有）依据先左后右的顺序加入s2，这样s2在访问的时候根据栈的先进后出原理就是右节点先访问，正好是偶数层需要的从右到左访问次序。偶数层则正好相反，要将子节点（如果有）依据先右后左的顺序加入s1，这样在s1访问的时候根据栈的先进后出原理就是左节点先访问，正好是奇数层需要的从左到右访问次序。
  • 每次访问完一层，即一个栈为空，则将一维数组加入二维数组中，并清空以便下一层用来记录。

# 📝代码实现

import java.util.*;
public class Solution {
    public ArrayList<ArrayList<Integer> > Print(TreeNode pRoot) {
        TreeNode head = pRoot;
        ArrayList<ArrayList<Integer> > res = new ArrayList<ArrayList<Integer>>();
        if(head == null)
            //如果是空，则直接返回空list
            return res; 
        //队列存储，进行层次遍历
        Queue<TreeNode> temp = new LinkedList<TreeNode>(); 
        temp.offer(head);
        TreeNode p;
        boolean flag = true;
        while(!temp.isEmpty()){
            //记录二叉树的某一行
            ArrayList<Integer> row = new ArrayList<Integer>();  
            int n = temp.size();
            //奇数行反转，偶数行不反转
            flag = !flag; 
            //因先进入的是根节点，故每层节点多少，队列大小就是多少
            for(int i = 0; i < n; i++){
                p = temp.poll();
                row.add(p.val);
                //若是左右孩子存在，则存入左右孩子作为下一个层次
                if(p.left != null)
                    temp.offer(p.left);
                if(p.right != null)
                    temp.offer(p.right);
            }
            //奇数行反转，偶数行不反转
            if(flag)  
                Collections.reverse(row);
            res.add(row);
        }
        return res;
    }
}

import java.util.*;
public class Solution {
    public ArrayList<ArrayList<Integer> > Print(TreeNode pRoot) {
       TreeNode head = pRoot;
       ArrayList<ArrayList<Integer> > res = new ArrayList<>();
       if(null == head){
           return res;
       }
       //定义两个栈 s1奇数栈 s2偶数栈
       Stack<TreeNode> s1 = new Stack<>();
       Stack<TreeNode> s2 = new Stack<>();
       s1.add(head);
       //当两个栈不同时为空时
       while(!s1.isEmpty() || !s2.isEmpty()){
           ArrayList<Integer> row = new ArrayList<>();
           //s1不为空时
           while(!s1.isEmpty()){
               TreeNode temp = s1.pop();
               row.add(temp.val);
               //子节点放到另一个栈
               if(null != temp.left){
                   s2.push(temp.left);
               }
               if(null != temp.right){
                   s2.push(temp.right);
               }
           }
           if(0 != row.size()){
               res.add(new ArrayList<Integer>(row));
           }
           //清空数组
           row.clear();
           //s2不为空时
           while(!s2.isEmpty()){
               TreeNode temp = s2.pop();
               row.add(temp.val);
               if(null != temp.right){
                   s1.push(temp.right);
               }
               if(null != temp.left){
                   s1.push(temp.left);
               }  
           }
           if(0 != row.size()){
               res.add(new ArrayList<Integer>(row));
           }
           row.clear();
       }
       return res;
    }
}

# 6、二叉树的最大深度

# ❓题目描述

求给定二叉树的最大深度， 深度是指树的根节点到任一叶子节点路径上节点的数量。 最大深度是所有叶子节点的深度的最大值。 （注：叶子节点是指没有子节点的节点。）

# 👉题目示例

示例1

输入：{1,2}
返回值：2
1
2

示例2

输入：{1,2,3,4,#,#,5}
返回值：3
1
2

# 💡题目解析

思路1：递归

如果根结点为空，则最大深度为0； 计算左子树的最大深度； 计算右子树的最大深度； 当前树的最大深度=左子树的最大深度和右子树的最大深度中的较大者+1

思路2：层次遍历 每遍历一层，深度+1

# 📝代码实现

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 * }
 */

public class Solution {
    /**
     * 
     * @param root TreeNode类 
     * @return int整型
     */
    public int maxDepth (TreeNode root) {
        // write code here
        if(null == root){
            return 0;
        }
        int maxLeft = 0, maxRight = 0;
        if(null != root.left){
            maxLeft = maxDepth(root.left);
        }
        if(null != root.right){
            maxRight = maxDepth(root.right);
        }
        int maxDepth = maxLeft > maxRight ? maxLeft : maxRight;
        return maxDepth + 1;
    }
}

思路2：层次遍历

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 * }
 */

public class Solution {
    /**
     * 层次遍历获取深度
     * @param root TreeNode类 
     * @return int整型
     */
    public int maxDepth (TreeNode root) {
        // write code here
        if(null == root){
            return 0;
        }
        //初始化存储队列，加入根节点
        Queue<TreeNode> node = new LinkedList<>();
        node.offer(root);
        //初始化返回结果
        int res = 0;
        while(!node.isEmpty()){
            //获取队列中节点个数
            int n = node.size();
            for(int i = 0; i < n; i++){
                TreeNode temp = node.poll();
                if(null != temp.left){
                    node.offer(temp.left);
                }
                if(null != temp.right){
                    node.offer(temp.right);
                }
            }
            res++;
        }
        return res;
    }
}

# 💡题目解析

思路1：递归

  • 首先，判断树节点是否为空，为空直接返回0
  • 再检查遍历到是否为叶子节点，且当前sum值等于节点值，说明可以刚好找到
  • 检查左右子节点是否可以有完成路径的，如果任意一条路径可以都返回true，因此这里选用两个子节点递归的或。

思路2：深度优先搜索（dfs）

  • 首先检查空节点，空树没有路径
  • 使用两个栈同步遍历，一个栈记录节点，辅助深度优先搜索，记录节点和，另一个栈跟随记录到该节点为止的路径和（C++中可以在一个栈中嵌套pair实现）。根节点及根节点值先进栈

  • 没有到叶子节点就将左右子节点（如果有）加入栈中，并跟随加入路径和
  • 如果遍历结束也没有找到路径和，则该二叉树中没有

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;

public class Solution {
    /**
     * 
     * @param root TreeNode类 
     * @param sum int整型 
     * @return bool布尔型

    public boolean hasPathSum (TreeNode root, int sum) {
        // 是否为空

            return true;
        }
        //递归进入子节点
        return hasPathSum(root.left, sum-root.val) || hasPathSum(root.right, sum - root.val);
    }
}

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 * }
 */

public class Solution {
    /**
     * 
     * @param root TreeNode类 
     * @param sum int整型 
     * @return bool布尔型
     */
    public boolean hasPathSum (TreeNode root, int sum) {
        // 是否为空
        if(null == root){
            return false;
        }
        //辅助栈 存储节点
        Stack<TreeNode> s1 = new Stack<TreeNode>();
        //辅助栈 存储和
        Stack<Integer> s2 = new Stack<Integer>();
        s1.add(root);
        s2.add(root.val);
        while(!s1.isEmpty()){
            //弹出栈节点
            TreeNode temp = s1.pop();
            int s_sum = s2.pop();
            if(null == temp.left && null == temp.right && s_sum == sum){
                return true;
            }
            //左子树节点
            if(null != temp.left){
                s1.add(temp.left);
                s2.add(s_sum + temp.left.val);
            }
            if(null != temp.right){
                s1.add(temp.right);
                s2.add(s_sum + temp.right.val);
            }
        }
        return false;
    }
}

# 💡题目解析

二叉搜索树最左端的元素一定最小，最右端的元素一定最大，符合“左中右”的特性，因此二叉搜索树的中序遍历就是一个递增序列，我们只要对它中序遍历就可以组装称为递增双向链表。

  • 创建两个指针，一个指向题目中要求的链表头（head），一个指向当前遍历的前一节点（pre)
  • 首先递归到最左，初始化head与pre
  • 然后处理中间根节点，依次连接pre与当前节点，连接后更新pre为当前节点
  • 最后递归进入右子树，继续处理
  • 递归出口即是节点为空则返回

public class TreeNode {

    TreeNode left = null;
    TreeNode right = null;

public class Solution {
    //返回的第一个指针，即为最小值，先定为null
    TreeNode head = null;
    //中序遍历当前值的上一位，初值为最小值，先定为null
    TreeNode pre = null;
    public TreeNode Convert(TreeNode pRootOfTree) {
        if(null == pRootOfTree){
            return null;
        }
        //递归到左子树到最小值
        Convert(pRootOfTree.left);
        //找到最小值，初始化head与pre
        if(null == pre){
            pre = pRootOfTree;
            head = pRootOfTree;
        }else{
            pre.right = pRootOfTree;
            pRootOfTree.left = pre;
            pre = pRootOfTree;
        }
        Convert(pRootOfTree.right);
        return head;
    }
}

public class Solution {

    public boolean rec(TreeNode r1, TreeNode r2){
        //遍历两个节点，同时为空，则是对称的
        if(null == r1 && null == r2){

    boolean isSymmetrical(TreeNode pRoot) {
        return rec(pRoot, pRoot);
    }

            if (temp1 == null || temp2 == null || temp1.val != temp2.val) {
                return false;
            }
            //从左往右加入队列
            q1.offer(temp1.left);
            q1.offer(temp1.right);
            //从右往左加入队列
            q2.offer(temp2.right);
            q2.offer(temp2.left);
        }
        return true;
    }
}

public class Solution {
    /**
     * 
     * @param t1 TreeNode类 
     * @param t2 TreeNode类 
     * @return TreeNode类
     */
    public TreeNode mergeTrees (TreeNode t1, TreeNode t2) {
        // 判断是否为空
        if(null == t1){
            return t2;
        }
        if(null == t2){
            return t1;
        }
        //获取头部
        TreeNode head = new TreeNode(t1.val + t2.val);
        //左子树节点
        head.left = mergeTrees(t1.left, t2.left);
        //右子树节点
        head.right = mergeTrees(t1.right, t2.right);
        return head;
    }
}

        Stack<TreeNode> s = new Stack<TreeNode>();
        //加入根节点
        s.push(pRoot);
        while(!s.isEmpty()){
            //弹出栈元素
            TreeNode temp = s.pop();
            //左右节点分别入栈
            if(null != temp.left){
                s.push(temp.left);
            }
            if(null != temp.right){
                s.push(temp.right);
            }
            //交换左右节点的值
            TreeNode tail = temp.left;
            temp.left = temp.right;
            temp.right = tail;
        }
        return pRoot;
}

# 12、判断是不是二叉搜索树

# ❓题目描述

给定一个二叉树根节点，请你判断这棵树是不是二叉搜索树。

二叉搜索树满足每个节点的左子树上的所有节点均小于当前节点且右子树上的所有节点均大于当前节点。

# 💡题目解析

思路：栈

  • 首先判断是否为空，为空返回true
  • 创建一个数组记录中序遍历的值
  • 创建一个辅助栈，用于存储左子树节点
  • 从根节点开始，每次优先进入每棵的子树的最左边一个节点，我们将其不断加入栈中，用来保存父问题
  • 到达最左后，可以开始访问，如果它还有右节点，则将右边也加入栈中，之后右子树的访问也是优先到最左
  • 遍历数组，依次比较相邻两个元素是否为递增序

import java.util.*;
public class Solution {
    public boolean isValidBST(TreeNode root){
        //设置栈用于遍历
        Stack<TreeNode> s = new Stack<TreeNode>(); 
        TreeNode head = root;
        //记录中序遍历的数组
        ArrayList<Integer> sort = new ArrayList<Integer>(); 
        while(head != null || !s.isEmpty()){
            //直到没有左节点
            while(head != null){   
                s.push(head);
                head = head.left;
            }
            head = s.pop();
            //访问节点
            sort.add(head.val); 
            //进入右边
            head = head.right; 
        }
        //遍历中序结果
        for(int i = 1; i < sort.size(); i++){ 
            //一旦有降序，则不是搜索树
            if(sort.get(i - 1) > sort.get(i)) 
                return false;
        }
        return true;
    }
}

# 13、判断是不是完全二叉树

# ❓题目描述

给定一个二叉树，确定他是否是一个完全二叉树。

完全二叉树的定义：若二叉树的深度为 h，除第 h 层外，其它各层的结点数都达到最大个数，第 h 层所有的叶子结点都连续集中在最左边，这就是完全二叉树。（第 h 层可能包含 [1~2h] 个节点）

# 💡题目解析

完全二叉树的特征：

  • 任意一个节点的左子节点为空，右子节点不为空，则不是完全二叉树
  • 当前节点的左右子节点并不是一定都有，那么之后的节点为叶子节点，否则就不是完全二叉树

# 📝代码实现

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 *   public TreeNode(int val) {
 *     this.val = val;
 *   }
 * }
 */

public class Solution {
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param root TreeNode类 
     * @return bool布尔型
     */
    public boolean isCompleteTree (TreeNode root) {
        // 判断是否为空
        if(null == root){
            return true;
        }
        //创建辅助队列存储节点
        Queue<TreeNode> queue = new LinkedList<TreeNode>();
        queue.offer(root);
        //叶子节点标志
        boolean isLeaf = false;
        while(!queue.isEmpty()){
            TreeNode temp = queue.poll();
            //左子结点为空，右子节点不为空的情况
            if(null == temp.left && null != temp.right){
                return false;
            }
            //如果是叶子节点且 左右子节点并不完全存在的情况
            if(isLeaf && (null != temp.left || null != temp.right)){
                return false;
            }
            //加入左子节点
            if(null != temp.left){
                queue.offer(temp.left);
            }
            //右子节点不为空，加入右子节点，为空说明是叶子节点了
            if(null != temp.right){
                queue.offer(temp.right);
            }else{
                isLeaf = true;
            }
        }
        return true;
    }
}

# 14、判断是不是平衡二叉树

# ❓题目描述

输入一棵节点数为 n 二叉树，判断该二叉树是否是平衡二叉树。

在这里，我们只需要考虑其平衡性，不需要考虑其是不是排序二叉树

平衡二叉树（Balanced Binary Tree），具有以下性质：它是一棵空树或它的左右两个子树的高度差的绝对值不超过1，并且左右两个子树都是一棵平衡二叉树。

# 返回值描述：

输出一个布尔类型的值

# 题目解析

平衡二叉树的判断：

  • 若树为空，则是平衡二叉树
  • 若树不为空，则左右子树的高度差不大于1

即题目可转换为计算左右子树的深度

# 📝代码实现

public class Solution {
    public boolean IsBalanced_Solution(TreeNode root) {
        //定义全局返回值
        boolean[] res = new boolean[1];
        res[0] = true;
        getHeight(root, res);
        return res[0];
    }

    public int getHeight(TreeNode head, boolean[] res){
        if(null == head){
            return 0;
        }
        //获取左子树节点的高度
        int leftLevel = getHeight(head.left,  res);
        //获取右子树节点的高度
        int rightLevel = getHeight(head.right,  res);
        //左右子树相差值
        if(Math.abs(leftLevel - rightLevel) > 1){
            res[0] = false;
        }
        //+ 1是为了加上根节点的深度
        return Math.max(leftLevel, rightLevel) + 1;
    }
}

# 15、二叉搜索树的最近公共祖先

4.p、q 为不同节点且均存在于给定的二叉搜索树中。

  • q,p在根节点左侧，递归遍历根节点的左子树
  • q,p在根节点右侧，递归遍历根节点的右子树

# 📝代码实现

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 *   public TreeNode(int val) {
 *     this.val = val;
 *   }
 * }
 */

public class Solution {
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param root TreeNode类 
     * @param p int整型 
     * @param q int整型 
     * @return int整型
     */
    public int lowestCommonAncestor (TreeNode root, int p, int q) {
        // 判断是否为空
        if(null == root){
            return -1;
        }
        //p,q在节点的两边
        if((p >= root.val && q <= root.val ) || (p <= root.val && q >= root.val)){
            return root.val;
        }
        //q,p在节点的左子树
        if(p <= root.val && q <= root.val){
            return lowestCommonAncestor(root.left, p, q);
        }
        //p,q在节点的右子树上
        if(p >= root.val && q >= root.val){
            return lowestCommonAncestor(root.right, p, q);
        }
        return -1;
    }
}

# 16、在二叉树中找到两个节点的最近公共祖先

# ❓题目描述

给定一棵二叉树(保证非空)以及这棵树上的两个节点对应的val值 o1 和 o2，请找到 o1 和 o2 的最近公共祖先节点。

数据范围：树上节点数满足 1≤n≤10^5 , 节点值val满足区间 [0,n)

要求：时间复杂度 O(n)

注：本题保证二叉树中每个节点的val值均不相同。

如当输入{3,5,1,6,2,0,8,#,#,7,4},5,1时，二叉树{3,5,1,6,2,0,8,#,#,7,4}如下图所示：

所以节点值为5和节点值为1的节点的最近公共祖先节点的节点值为3，所以对应的输出为3。节点本身可以视为自己的祖先

# 💡题目解析

这里题目跟上一个有些类似，那同样其实也可以采用递归的方式解决

思路2：深度优先搜索

  • 创建一个深度优先搜索的方法，参数（TreeNode root, List list, int o），查找标志位flag = false
    • list添加当前节点的值，如果当前值 == o，则说明查找到了节点，返回，false = true;
    • 当前节点的左右子节点不为空，分别递归左右子节点
    • 最后，查找完了，要回溯到父节点，也就是移除刚刚加入的节点
  • 初始化存储左右子树路径的list，分别调用深度优先搜索方法
  • 遍历两条路径数组，依次比较元素值
  • 找到两条路径第一个不相同的节点即是最近公共祖先

# 📝代码实现

思路1：递归实现

import java.util.*;
public class Solution {
    public int lowestCommonAncestor (TreeNode root, int o1, int o2) {
        //该子树没找到，返回-1
        if(root == null) 
            return -1;
        //该节点是其中某一个节点
        if(root.val == o1 || root.val == o2) 
            return root.val;
        //左子树寻找公共祖先
        int left = lowestCommonAncestor(root.left, o1, o2); 
        //右子树寻找公共祖先
        int right = lowestCommonAncestor(root.right, o1, o2); 
        //左子树为没找到，则在右子树中
        if(left == -1) 
            return right;
        //右子树没找到，则在左子树中
        if(right == -1) 
            return left;
        //否则是当前节点
        return root.val; 
    }
}

思路2：深度优先搜索

import java.util.*;

/*
 * public class TreeNode {
 *   int val = 0;
 *   TreeNode left = null;
 *   TreeNode right = null;
 * }
 */

public class Solution {
    /**
     * 
     * @param root TreeNode类 
     * @param o1 int整型 
     * @param o2 int整型 
     * @return int整型
     */
     boolean flag = false;
    public int lowestCommonAncestor (TreeNode root, int o1, int o2) {
        // 初始化左右子树的list
        List<Integer> list1 = new ArrayList<Integer>();
        List<Integer> list2 = new ArrayList<Integer>();
        dfs(root, list1, o1);
        //重置flag
        flag = false;
        //遍历查询o2
        dfs(root, list2, o2);
        int res = 0;
        for(int i = 0; i < list1.size() && i < list2.size(); i++){
            int m = list1.get(i);
            int n = list2.get(i);
            //最近相同点就是 最近公共祖先
            if(m == n){
                res = n;
            }else{
                break;
            }
        }
        return res;
    }

    public void dfs(TreeNode root, List<Integer> list, int o){
        //如果为空，则返回
        if(null == root){
            return;
        }
        //增加当前节点的值
        list.add(root.val);
        //如果查找到了当前值
        if(root.val == o){
            flag = true;
            return;
        }
        if(null != root.left){
            dfs(root.left, list, o);
        }
        if(null != root.right){
            dfs(root.right, list, o);
        }
        if(flag){
            return;
        }
        //移除当前节点
        list.remove(list.size() - 1);
    }
}
