# 1、寻找数组的中心索引

力扣链接：https://leetcode-cn.com/leetbook/read/array-and-string/yf47s/

给你一个整数数组 nums ，请计算数组的 中心下标 。

数组 中心下标 是数组的一个下标，其左侧所有元素相加的和等于右侧所有元素相加的和。

如果中心下标位于数组最左端，那么左侧数之和视为 0 ，因为在下标的左侧不存在元素。这一点对于中心下标位于数组最右端同样适用。

如果数组有多个中心下标，应该返回 最靠近左边 的那一个。如果数组不存在中心下标，返回 -1 。

示例 1：

输入：nums = [1, 7, 3, 6, 5, 6]
输出：3
解释：
中心下标是 3 。
左侧数之和 sum = nums[0] + nums[1] + nums[2] = 1 + 7 + 3 = 11 ，
右侧数之和 sum = nums[4] + nums[5] = 5 + 6 = 11 ，二者相等。

public static void main(String[] args) {
    int[] nums = {1, 7, 3, 6, 5, 6};
    System.out.println(centerSubscript(nums));

/**
     * 1、记数组的全部元素之和为 total，当遍历到第 i个元素时，设其左侧元素之和为sum，则其右侧元素之和为total-sum-nums[i]
     * 2、左侧元素之和 = 右侧元素之和
     * 3、sum = total-sum-nums[i]
     * 4、2*sum + nums[i] = total
     * @param nums
     * @return
     */
public static int centerSubscript(int []nums){
    //计算数组所有的值的和
    int total = Arrays.stream(nums).sum();
    int sum = 0;
    for (int i = 0; i < nums.length; ++i) {
        if(2*sum + nums[i] == total){
            return i;
        }
        sum += nums[i];
    }
    return -1;
}

# 1、设计链表

设计链表的实现。您可以选择使用单链表或双链表。单链表中的节点应该具有两个属性：val 和 next。val 是当前节点的值，next 是指向下一个节点的指针/引用。如果要使用双向链表，则还需要一个属性 prev 以指示链表中的上一个节点。假设链表中的所有节点都是 0-index 的。

在链表类中实现这些功能：

get(index)：获取链表中第 index 个节点的值。如果索引无效，则返回-1。 addAtHead(val)：在链表的第一个元素之前添加一个值为 val 的节点。插入后，新节点将成为链表的第一个节点。 addAtTail(val)：将值为 val 的节点追加到链表的最后一个元素。 addAtIndex(index,val)：在链表中的第 index 个节点之前添加值为 val 的节点。如果 index 等于链表的长度，则该节点将附加到链表的末尾。如果 index 大于链表长度，则不会插入节点。如果index小于0，则在头部插入节点。 deleteAtIndex(index)：如果索引 index 有效，则删除链表中的第 index 个节点。

public class ListNode{
    int val;
    ListNode next;
    ListNode(int x) { val = x; }

}

class MyLinkedList {
    int size;
    //头结点
    ListNode head;

    /** 初始化链表 */
    public MyLinkedList() {
        head = new ListNode(0);
        size = 0;
    }
    
    /** 获取指定位置元素 */
    public int get(int index) {
        if (index < 0 || index >= size) return -1;
        ListNode cur = head;
        for(int i = 0; i <= index; i++){
            cur = cur.next;
        }
        return cur.val;
    }
    
    /** 头结点增加元素 */
    public void addAtHead(int val) {
        addAtIndex(0,val);
    }
    
    /** 尾节点增加元素*/
    public void addAtTail(int val) {
        addAtIndex(size,val);
    }
    
    /** 指定位置增加元素 */
    public void addAtIndex(int index, int val) {
        if(index > size){
            return;
        }
        if (index < 0) index = 0;
        ListNode pre = head;
        for(int i = 0; i < index; i++){
            pre = pre.next;
        }
        ListNode ln = new ListNode(val);
        ln.next = pre.next;
        pre.next = ln;
        size++;
    }
    
    /** 删除指定位置元素*/
    public void deleteAtIndex(int index) {
        if (index < 0 || index >= size) return;

        size--;
        ListNode pred = head;
        for(int i = 0; i < index; ++i) pred = pred.next;
        pred.next = pred.next.next;
    }
}

# 2、双指针问题

判断是否有环：一个安全的选择是每次移动慢指针一步，而移动快指针两步。每一次迭代，快速指针将额外移动一步。如果环的长度为 M，经过 M 次迭代后，快指针肯定会多绕环一周，并赶上慢指针。

# 1、环形链表

给定一个链表，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。 为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。注意：pos 不作为参数进行传递，仅仅是为了标识链表的实际情况。

如果链表中存在环，则返回 true 。 否则，返回 false 。

作者：力扣 (LeetCode) 链接：https://leetcode-cn.com/leetbook/read/linked-list/jbex5/ 来源：力扣（LeetCode） 著作权归作者所有。商业转载请联系作者获得授权，非商业转载请注明出处。

双指针解法

假想「乌龟」和「兔子」在链表上移动，「兔子」跑得快，「乌龟」跑得慢。当「乌龟」和「兔子」从链表上的同一个节点开始移动时，如果该链表中没有环，那么「兔子」将一直处于「乌龟」的前方；如果该链表中有环，那么「兔子」会先于「乌龟」进入环，并且一直在环内移动。等到「乌龟」进入环时，由于「兔子」的速度快，它一定会在某个时刻与乌龟相遇，即套了「乌龟」若干圈。

我们可以根据上述思路来解决本题。具体地，我们定义两个指针，一快一满。慢指针每次只移动一步，而快指针每次移动两步。初始时，慢指针在位置 head，而快指针在位置 head.next。这样一来，如果在移动的过程中，快指针反过来追上慢指针，就说明该链表为环形链表。否则快指针将到达链表尾部，该链表不为环形链表

/**
 * Definition for singly-linked list.
 * class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public boolean hasCycle(ListNode head) {
        if(null == head || null == head.next){
            return false;
        }
        ListNode fast = head;
        ListNode low = head;
        while(null !=fast && null != fast.next){
            low = low.next;
            fast = fast.next.next;
            if(fast == low){
                return true;
            }
        }
        return false;
    }
}

哈希表解法

最容易想到的方法是遍历所有节点，每次遍历到一个节点时，判断该节点此前是否被访问过。

具体地，我们可以使用哈希表来存储所有已经访问过的节点。每次我们到达一个节点，如果该节点已经存在于哈希表中，则说明该链表是环形链表，否则就将该节点加入哈希表中。重复这一过程，直到我们遍历完整个链表即可。

public class Solution {
    public boolean hasCycle(ListNode head) {
        //set不能存储重复的值
        Set<ListNode> seen = new HashSet<ListNode>();
        while (head != null) {
            //把节点值存入set集合中
            if (!seen.add(head)) {
                return true;
            }
            head = head.next;
        }
        return false;
    }
}

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }

 */
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        Set<ListNode> forSet = new HashSet<ListNode>();
        ListNode temp = headA;
        while(null != temp){
            forSet.add(temp);
            temp = temp.next;
        }
        temp = headB;
        while(null != temp){
            if(forSet.contains(temp)){
                return temp;
            }
            temp = temp.next;
        }
        return null;
    }
}

双指针

使用双指针的方法，可以将空间复杂度降至 O(1)O(1)。

只有当链表headA 和headB 都不为空时，两个链表才可能相交。因此首先判断链表headA 和 headB 是否为空，如果其中至少有一个链表为空，则两个链表一定不相交，返回 null。

当链表 headA 和 }headB 都不为空时，创建两个指针pA 和 pB，初始时分别指向两个链表的头节点 headA 和 headB，然后将两个指针依次遍历两个链表的每个节点。具体做法如下：

每步操作需要同时更新指针pA 和pB。

如果指针pA 不为空，则将指针pA 移到下一个节点；如果指针pB 不为空，则将指针pB 移到下一个节点。

如果指针pA 为空，则将指针pA 移到链表headB 的头节点；如果指针pB 为空，则将指针pB 移到链表 headA 的头节点。

当指针pA 和pB 指向同一个节点或者都为空时，返回它们指向的节点或者 \text{null}null。

public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        if (headA == null || headB == null) {
            return null;
        }
        ListNode pA = headA, pB = headB;
        while (pA != pB) {
            pA = pA == null ? headB : pA.next;
            pB = pB == null ? headA : pB.next;
        }
        return pA;
    }
}

        int length = 0;

            head = head.next;
        }
        return length;
    }
}

# 5、奇偶链表

给定单链表，将所有奇数节点连接在一起，然后将偶数节点连接在一起。 请注意，这里我们讨论的是节点编号，而不是节点中的值。

样例1:

输入： 1->2->3->4->5->NULL
输出： 1->3->5->2->4->NULL
1
2

样例2:

输入： 2->1->null
输出： 2->1->null
1
2

思路：定义两个指针来分别存奇数和偶数编号的节点，遍历完之后，奇数链表的下一个节点指向偶数链表

public class Solution {
    /**
     * @param head: a singly linked list
     * @return: Modified linked list
     */
    public ListNode oddEvenList(ListNode head) {
        //如果链表为空或者只有一个节点时
        if(null == head || null == head.next){
            return head;
        }
        //奇偶数链表来接收编号的值
        ListNode odd = head, even = head.next;
        //从偶数编号开始
        ListNode temp = even;
        while(null != even && null != even.next){
            //奇数编号指向偶数编号的下一个节点
            odd.next = even.next;
            //重新赋值
            odd = odd.next;
            //偶数编号的下一个节点指向后面第二个节点
            even.next = even.next.next;
            //重新赋值
            even = even.next;
        }
        //奇数编号链表指向偶数链表
        odd.next = temp;
        return head;
    }
}

输入：s = "()"
输出：true

示例 2：

输入：s = "()[]{}"
输出：true

输入：s = "(]"
输出：false

示例 4：

输入：s = "([)]"

示例 5：

输入：s = "{[]}"

思路：

  1. 把左括号存入栈中
  2. 当遇到一个右括号时，这时取出栈顶元素是否为相同类型的括号
    • 如果不是，或者栈没有左括号，则不是有效的括号
  3. 用哈希表存储每一种括号，键为右括号，值为左括号
  4. 如果括号数为奇数，则也不是有效的括号
public boolean isValid(String s) {
    if(s.length() % 2 == 1){
        return false;
    }
    //存储括号
    Map<Character,Character> map = new HashMap<Character,Character>(){{
        put(')','(');
        put(']','[');
        put('}','{');
    }};
    //存储左括号栈
    Deque<Character> stack = new LinkedList<Character>();
    for(int i = 0; i < s.length(); i++){
        char ch = s.charAt(i);
        //如果哈希表中包含key，则弹；，不包含，则存入左括号
        if(map.containsKey(ch)){
            if (stack.isEmpty() || stack.peek() != map.get(ch)) {
                return false;
            }
            stack.pop();
        }else{
            stack.push(ch);
        }
    }
    return stack.isEmpty();
}

思路：利用for循环一次向后遍历，每次找到比当前值大的值，然后用下标相减

public int[] dailyTemperatures(int[] temperatures) {
    int n = temperatures.length;
    int[] res = new int[n];
    for(int i = 0; i < n; i++){
        int curr = temperatures[i];
        for(int j = i + 1; j < n; j++){
            if(curr < temperatures[j]){
                res[i] = j - i;
                break;
            }
        }
    }
    return res;
}

思路：定义一个栈来存储下标，如果比较的元素大于栈顶的下标所在元素，则弹出该下标，

public int[] dailyTemperatures(int[] temperatures) {
    int n = temperatures.length;
    int[] res = new int[n];
    //存储下标
    Deque<Integer> stack = new LinkedList<Integer>();
    for(int i = 0; i < n; i++){
        int temp = temperatures[i];
        while(!stack.isEmpty() && temp > temperatures[stack.peek()]){
            //弹出上一个下标
            int preIndex = stack.pop();
            res[preIndex] = i - preIndex; 
        }
        stack.push(i);
    }
    return res;

    int n1 = nums1.length;

}

# 2、栈

思路：

nums1 = [4,1,2], nums2 = [1,3,4,2]

用Map键存储目标值，值存储第一个比目标值大的数，例如（1，3），(3，4)

对于数字x，若栈顶元素小于x，则将栈顶元素的答案置为x，直到栈顶大于元素x，此时将x入栈，重复此过程从左到右遍历nums2即可得到所有数字的答案。

然后遍历nums1，如果找到map中有对应的值，就返回默认值，没有则为-1

public int[] nextGreaterElement(int[] nums1, int[] nums2) {
    Map<Integer, Integer> map = new HashMap<>();
    Stack<Integer> stack = new Stack<>();
    for (int num : nums2) {
        while (!stack.isEmpty() && stack.peek() < num) {
            map.put(stack.pop(), num);
        }
        stack.push(num);
    }
    for (int i = 0; i < nums1.length; i++) {
        nums1[i] = map.getOrDefault(nums1[i], -1);
    }
    return nums1;
}
