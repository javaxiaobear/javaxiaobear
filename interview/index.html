Java基础

要了解一下Java概述，熟悉基本语法、数据类型、异常的处理，重点是面向对象，封装、继承、多态；面试中高频问点就是多线程与并发、JVM，还有集合的源码等

Java高级

高级部分包括Java反射，设计模式以及类加载器等

Java Web

Java Web可以说是Java程序员Web端的入门，虽然近几年基本上很少使用jsp等技术来进行开发，但我们还是要掌握xml、cookie、session、jdbc等常用的技术

数据库

数据库主要以MySQL、Redis为主，这两个基本上是主要问的

Java开发框架

框架这部分包含很多，入门框架有Spring、Spring MVC、Mybatis，目前SSH很少使用了，微服务生态这边有Spring Boot、Spring Cloud
